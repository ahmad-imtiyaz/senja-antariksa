# **Senja Antariksa**

## **Project Overview**

**Senja Antariksa** adalah website portofolio personal yang dirancang dengan pendekatan visual modern dan nuansa estetika yang tenang. Project ini berfungsi sebagai media untuk menampilkan profil, karya, serta informasi profesional dalam satu halaman web yang terstruktur dan mudah diakses.

Website ini menekankan pada kesederhanaan desain, keterbacaan konten, serta responsivitas tampilan di berbagai perangkat.

## **Features**

Fitur utama yang tersedia pada project ini:

* Landing page dengan desain minimalis
* Informasi profil dan deskripsi singkat
* Bagian portofolio atau karya
* Navigasi sederhana dan intuitif
* Tampilan responsif untuk desktop dan mobile

## **Tech Stack**

Project ini dibangun menggunakan teknologi berikut:

* **HTML5** — struktur konten website
* **CSS3** — styling dan layout
* **JavaScript** — interaksi dasar
* **Bootstrap** — framework UI responsif
* **Git & GitHub** — version control

## **Installation & Setup**

1. Clone repository:

   ```bash
   git clone https://github.com/ahmad-imtiyaz/senja-antariksa.git
   ```

2. Masuk ke direktori project:

   ```bash
   cd senja-antariksa
   ```

3. Jalankan project:

   * Buka file `index.html` langsung di browser
     **atau**
   * Gunakan local server (misalnya Live Server di VS Code)

## **Usage**

Project ini dapat digunakan sebagai:

* Template portofolio personal
* Referensi desain website statis
* Media presentasi profil dan karya

Konten dapat disesuaikan dengan mengubah file HTML dan CSS sesuai kebutuhan.

## **Folder Structure**

Struktur direktori utama:

```
/
├── assets/           # Gambar dan aset pendukung
├── css/              # File styling
├── js/               # Script JavaScript
├── index.html        # Halaman utama
└── README.md         # Dokumentasi project
```
